DEMO OF 3D and 4D RECONSTRUCTION OF TOMOGRAPHIC DATA.

******* WHAT YOU HAVE TO DO ********
- Install the following software dependencies if not already installed.
	- MPI compiler (openmpi, intel mpi, etc.)
	- Open MP 
	- Make utility

******* HOW TO PERFORM RECONSTRUCTION *****
3D Reconstruction of Shepp-Logan Phantom
- Unix/Linux/Mac OS : Go to the folder '/demo/recon_3d/shepp-logan-3D/' and run the script 'run_unix.sh'
- Super-computing cluster (Rice cluster at Purdue): Go to the folder '/demo/recon_3d/shepp-logan-3D/' and run the script 'run_cluster.sh' 

4D Reconstruction of Cahn-Hilliard Phantom
- Unix/Linux/Mac OS : Go to the folder '/demo/recon_4d/cahn-hilliard-4D/' and run the script 'run_unix.sh'
- Super-computing cluster (Rice cluster at Purdue): Go to the folder '/demo/recon_4d/cahn-hilliard-4D/' and run the script 'run_cluster.sh'
