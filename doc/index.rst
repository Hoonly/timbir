.. title:: TIMBIR

The Time-Interlaced Model-Based Iterative Reconstruction (`TIMBIR <https://github.com/adityamnk/timbir.git>`_) 
is a method for 4D time-space reconstruction of data acquired using synchrotron X-ray computed tomography.

This guide is maintained on 
`GitHub <https://github.com/adityamnk/timbir/tree/master/doc>`_.
